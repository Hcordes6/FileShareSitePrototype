## File Sharing News Site made for full-stack development class

Website made from scratch, supporting user registration/login, creating news posts for authenticated users, Upvote/downvote system, + more. 

# Tech Stack
- HTML/CSS Frontend
- PHP Backend
- MySQL Database
- AWS Cloud Instance
- Apache Web Server
